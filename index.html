This tracker is only available between 7:00am and 8:00pm, Eastern Standard Time.

https://stm.austinhuang.me / https://stmbus.glitch.me

Tracks STM "École" buses. Provides real-time location, stop sequence, arrival predictions and comparison to scheduled time, through STM's GTFS-Realtime API as well as my own interpretations of school-provided bus schedules.

This unofficial service is voluntarily offered to the following cégeps:

  • Marianopolis 124E (All features)
  • André-Laurendeau 113E (Just raw data for now, full support soon)
  • Collège de Rosemont 18E (Just raw data)

As well as the following high schools:

  • LaSalle Comprehensive / PACC 112E (Just raw data)

Coming soon™: Collège St-Jean-Vianney, Collège Beaubois, Collège de Montréal

Read the change log!

Notes:

  • This tracker pings the STM API every 45 seconds.
  • This tracker is NOT accessible between 8:00pm and 7:00am. This is necessary to respect the API ratelimits imposed by the STM.
  • Some information are guessed. STM does not publicize trip IDs for school shuttles, which requires me to manually figure them out. If there are any inaccuracies, report them to me.
  • If you want your school to be added, please send your school's bus schedule, stops, and route number to me.
  • Privacy Policy: I do not store any data related to you. I do not even keep logs. I have also listed all the third-party privacy policies that you could be subjected at the bottom of every page.
  • Terms of Service: If you are not a student at the concerning institutions, please do NOT use this tool to get shortcuts for your commute. As well, please respect all laws and third-party obligations, listed at the bottom of every page.
  • This tracker is open source*.
