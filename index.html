This tracker is only available between 7:00am and 8:00pm, Eastern Standard Time.

Track STM "École" buses.

This unofficial service is offered to the following cégeps:

  • Marianopolis 124E (Includes direction & stop interpreter)
  • André-Laurendeau 113E (Just raw data)
  • Collège de Rosemont 18E (Just raw data)

Notes:

  • This tracker pings the STM API every 45 seconds.
  • This tracker will NOT update between 8:00pm and 7:00am. This is necessary to respect the API ratelimits imposed by the STM.
  • Trip IDs may be guessed. If there is any inaccuracies regarding the direction of a bus ("It's going to the metro station but tracker says it's going to the school???"), report them to me.
  • I'm responsible enough to list all the third-party privacy policies that you could be subjected at the bottom of every page. You think I'm gonna track you down while I'm still struggling with R-score??? (No, seriously, read the source code.)
